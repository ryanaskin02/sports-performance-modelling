# Sports Performance Modelling

Repository for exploratory work in statistical modelling and data science applied to sports performance.

## Directory Structure 

- `01_reading` A place to keep and share relevant readings, papers and my notes on them.
- `02_working_notebooks` A place to keep rough write-ups as I go.
- `03_code/` A place to keep all data, source files and output for code relating to the project. 

- `.gitignore` file names and types here that I do not want to version control.
